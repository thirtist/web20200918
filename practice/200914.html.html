Element

Element는 <시작태그>컨탠트(내용)<종료태그>로 이루어져있다

이 중에서 종료태그가 없는 img,br같은 empty element도 존재한다(XHTML에서는 시작태그안에 /를 사용해서 종료시켜줘야함)

컨탠트(내용)에 특수기호를 써야되는 경우 html entity를 구글에 검색해라(<> 이거 같은거)

block element와 inline element

block element는 줄을 모두 차지하고 inline element는 줄을 차지하지 않아 줄바꿈이 일어나지 않는다
h1등은 블록이고 p랑 a는 인라인 엘레먼트이다

attribute는 알맞는 element와 사용해야 한다

그림링크

A~href는 E~a와 함께 써야 작동한다(E~h1에 A~herf넣으면 동작안함-w3schools에서 레퍼런스를 참고하자)

주요 Element

br - 줄바꿈,empty element

pre - html에서 작성한대로 띄어쓰기 줄바꿈을 무시하지 않고 그대로 구현

head 내의 style - css에서 다루는 내용이며 선택자(id,class,E~)를 활용한다

Attribute

Element를 세부적으로 나타내는 속성이며, Attribute이름="proverty"으로 이루어져있다

Property와 Value를 가지며, ="property:value";형식으로 묶여있다(ex style="color: blue;")

style attribute

inline css로써 element옆에 삽입하여 사용한다(이 때는 적용 우선순위가 selector(선택자) id보다 높음)

style은 css에서 자세히 다루기에 head부분에서 style element로도 활용된다(외부 css는 head부분에서 link element를 사용)

주요 attribute

title - 툴팁(마우스가져가대면나오는설명) - 글로벌 attribute로 모든 element랑 사용가능

Property:Value

=":"로 묶여있는 것 주의

Color property

Color의 value는 rgb(255,255,255,0-마지막이건 투명도)로 지정
