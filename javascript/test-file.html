서버에서 할 것

git scm설치후 git bash실행
git config --global user.name 아이디
git config --global user.email 이메일
소스컨트롤 위에 commit을 눌러 저장함(스냅샷방식으로 스냅이름적기)
git bash에서 저장폴더를 찾아감(cd Documents/myworkspace/web/)
알맞게 찾아가면 (master)라는 글자가 보임

깃허브에 처음올리기

사이트에서 new repository만들면 아래에
git remote add origin 주소.git를 복사
bash에서 쉬프트+insert키로 붙여넣고 엔터
git push~부분도 똑같이 복붙해서 입력
로긴화면에서 로긴하면 업로드됨
이제 파일수정마다 commit하고 push로 올리기/pull로 받기

진짜 집에서 작업함

git scm 설치후 git bash실행
비쥬얼스튜디오코드에서 myworkspace만들기
창이 뜨면 git config --global user.name 아이디
git config --global user.email 이메일 입력
myworkspace에서 마우스 우클-bash here클릭
git사이트에서 주소 복사해서 git clone (쉬프트+inser로 붙여넣기) web 엔터누르기(웹은 폴더이름)
이제 비쥬얼스튜디오코드에서 commit으로 저장하고 push로 올리기/또는 서버에서 pull로 받기
